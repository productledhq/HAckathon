## Design System

### Colors
- Background: `#1A1A1A`
- Card: `#202020`
- Border: `#2A2A2A`
- Text: `#F7F7F7`
- Primary: `#3B82F6` (blue)
- Success: `#22C55E` (green)
- Warning: `#EAB308` (yellow)
- Danger: `#EF4444` (red)

### ICP Score Colors
- **High (85+)**: Green `#22C55E` - Great voice match
- **Medium (60-84)**: Yellow `#EAB308` - Needs improvement
- **Low (< 60)**: Red `#EF4444` - Critical issues

### Workflow Statuses
1. **Idea** (draft) - Gray - Not started
2. **In Progress** - Blue - Being edited
3. **Ready for Review** (pending) - Yellow - Awaiting approval
4. **Approved** - Green - Ready to schedule
5. **Scheduled** - Purple - Scheduled for publishing
6. **Published** - Green - Live content
7. **Rejected** - Red - Needs rewrite
